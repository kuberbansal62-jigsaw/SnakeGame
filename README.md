# SnakeGame
# 🐍 Snake Game in Python (Turtle)

A classic **Snake Game** built using Python's built-in **Turtle** graphics library. Guide the snake to eat food, grow longer, and achieve the highest score while avoiding collisions with the walls and your own tail.

## ✨ Features

* Classic Snake gameplay
* Built using Python's built-in `turtle` module
* Smooth keyboard controls
* Random food generation
* Live score display
* Collision detection with walls and snake body
* Simple, clean, and beginner-friendly code

## 📥 Installation

1. Clone the repository:

```bash
git clone https://github.com/kuberbansal62-jigsaw/Snakegame.git
```

2. Navigate to the project directory:

```bash
cd snake-game
```

3. Run the game:

```bash
python snake.py
```

> Replace `snake.py` with your main Python file if it has a different name.
